## 冒泡排序 ##

比较相邻的两个元素 如果前一个比后一个大那么交换位置 小的在前大的在后
```
function sort(arr){
    for(let i = 0;i<arr.length;i++){
        for(let j = 0;j<arr.length-i-1;j++){
            if(arr[j] > arr[j+1]){
                let temp = arr[j]
                arr[j] = arr[j+1]
                arr[j+1] = temp
            }
        }
    }
    return arr
}
console.log(sort([4,3,1,6,7,5]))
```
